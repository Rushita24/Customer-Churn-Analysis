# Customer Churn Analysis

## Business Objective
Analyze customer churn for a subscription-based company to identify key drivers and recommend retention strategies.

## Dataset
Telco Customer Churn dataset containing customer demographics, contract details, and churn status.

## Tools (Planned)
- SQL
- tableau
- Excel
## Dataset
- `data/Customer_Churn.csv.numbers`  
Contains customer demographics, contract details, and churn status for a subscription-based company.

## Key Insights
- Higher monthly charges correlate with churn
- Early tenure customers churn more
- Contract type strongly impacts retention
